Front End Starter Kit

Bootstrap based HTML, CSS Framework

Large scale responsive UI with ease

This is HTML, CSS framework for developing responsive, mobile first large scale projects on the web. It is based on Bootstrap-Sass, written in Sass and compiled with Gulp, extensively uses Flexible Box Layout Module and CSS Table display. This allows to develop large scale responsive UI of any complexity fast and easily.

Download GitHub

To use this framework you should be familiar with Bootstrap 3 and Bootstrap 4.

Installation

Clone or download.

$ cd front-end-starter-kit
$ npm install
$ npm start

This will compile css, js and html into the /dist and /public folders. /dist has minified versions of html, js and css, /public – mapped css for quick reference in Chrome Dev Tool.

Serve URL localhost:8080

Start with this basic HTML template client/index.html. Or copy the HTML below to begin working with a minimal document.

Icons

Here's 2 types of icons you can use, font icons and svg icons.

Font icons are very similar to Font Awesome, they have .fi class prefix and you will have to create your own font with FontForge, for example. This is not a trivial task but icons created this way are really good, easy to use and placed perfectly inside any elements. 

<span class="fi fi-bug fi-spin"></span> Some spinning bug…

SVG icons have icon class, supported well by major browsers, inherit text color and size but they are not perfectly placed vertically and looks like a mess inside your code.

Images & Video

Image classes extended with .img-fluid – 100% width image.

Images can be used as background pictures that cover or contain entire block element with specific ratio (16×9, 4×3, 3×2, 8×5, 1×1, etc.). Correspondent wrapper classes are used .img-contain and .img-cover with ratio class inside: 

<div class="bg-cover" style="background-image: url(image_to_cover)">
    <div class="img-ratio-16x9"></div>
</div>

Video frames have wrapper class .img-video-wrapper with iframe inside to make it 100% width and `auto` height. 

<div class="img-video-wrapper">
    <iframe src="…" frameborder="0"></iframe>
</div>
